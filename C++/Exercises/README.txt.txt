OLÁ,

PARA EXECUTAR O CÓDIGO É PRECISO ABRIR O CODEBLOCKS.
DENTRO DO CODEBLOCKS É PRECISO CLICAR EM FILE > OPEN > NAMESOBRENOME.CPP.
APÓS ABRIR O FICHEIRO É NECESSÁRIO EXECUTAR O CÓDIGO, PARA TAL PRECISA DE CLICAR NO BOTÃO 'BUILD AND RUN' QUE É O SIMBOLO DE UMA RODA DENTADA E UM BOTÃO DE 'PLAY' A AMARELO. ISTO EXECUTARÁ O CÓDGIO PARA VOCÊ UTILIZAR.

DENTRO DO CÓDIGO ESTÁ PRESENTE O CÓDIGO QUE PEDE AO UTILIZAR O NOME E O SOBRENOME.
DE SEGUIDA É APRESENTADO AO UTILIZADOR A PRIMEIRA INICIAL O JOGADOR COM O SOBRENOME. EX: H. FERREIRA;
DEPOIS É APRESENTADO A INICIAL DO SOBRENOME. EX: F;
SEGUIDO PELA INCIAL DO NOME. EX: H;
POR FIM, É APRESENTADO UM 'NICKNAME' CRIADO PARA O UTILIZADOR COM A INICIAL DE CADA NOME. EX: HF.

ADEUS!
